Simple AirBnB clone web page that uses server side JS to display information about properties from database to web pages via SQL queries.

## HOW TO USE

1. Clone repo
2. ```npm i```
3. ```nvm use 14```
4. ```npm run local```
5. go to ```localhost:3000``` on browser!
